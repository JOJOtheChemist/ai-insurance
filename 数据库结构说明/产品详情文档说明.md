# 保险产品详情文档说明

## 文档定义
**产品详情**是不包含任何客户个人信息的纯产品介绍文档,用于向潜在客户展示保险产品的功能、保障和特点。

---

## 一、数据来源

### 主表：insurance_product（保险产品表）

---

## 二、文档应包含的内容

### 1. 基础信息
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 产品名称 | product_name | ✅ | 保险产品的完整名称 |
| 产品代码 | product_code | ✅ | 产品唯一标识码 |
| 产品类型 | product_type | ✅ | 如:医疗险、重疾险、寿险等 |
| 保险公司 | company_name | ✅ | 承保公司名称 |
| 产品图片 | image_url | ⭕ | 产品宣传图/LOGO |
| 产品描述 | description | ✅ | 简短的产品卖点描述(约26字) |

### 2. 保费与保额信息
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 最低保额 | min_amount | ✅ | 最低保障金额 |
| 最高保额 | max_amount | ✅ | 最高保障金额 |
| 最低保费 | min_premium | ✅ | 起步价格 |
| 最高保费 | max_premium | ✅ | 最高价格 |

### 3. 投保规则
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 投保年龄 | age_range | ✅ | 如:0-65周岁 |
| 保险期间 | insurance_period | ✅ | 如:终身、至70周岁 |
| 交费期间 | payment_period | ✅ | 如:趸交、5年/10年/20年交 |
| 等待期 | waiting_period | ✅ | 如:90天、180天 |
| 犹豫期 | cooling_off_period | ✅ | 如:15天、20天 |

### 4. 保障内容
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 保障范围 | coverage | ✅ | 详细的保障项目列表(约222字) |
| 产品详情 | details | ⭕ | 更详细的产品介绍(TEXT类型) |
| 产品标签 | tags | ✅ | 关键字标签(约31字) |

### 5. 免责与退保
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 责任免除 | exclusions | ✅ | 不赔偿的情况说明(约434字) |
| 退保说明 | surrender_terms | ⭕ | 退保规则和损失说明 |

### 6. 扩展信息
| 字段 | 数据库字段 | 必填 | 说明 |
|------|-----------|------|------|
| 其他信息 | extend_info | ⭕ | JSON格式的其他必要信息 |

---

## 三、文档结构建议

```
【产品详情】友邦XX保险产品

一、产品概述
  - 产品名称、代码、类型
  - 保险公司
  - 产品描述、标签

二、投保须知
  - 投保年龄范围
  - 保险期间
  - 交费期间
  - 等待期/犹豫期

三、保障内容
  - 保额范围(最低-最高)
  - 保费范围(最低-最高)
  - 详细保障范围
  - 产品详情

四、重要声明
  - 责任免除
  - 退保说明

五、附加信息
  - 其他扩展信息
```

---

## 四、注意事项

### ✅ 应该包含的内容
- 产品功能和特性
- 保障范围和金额
- 投保规则和限制
- 费率参考范围
- 免责条款
- 产品标签和关键词

### ❌ 不应该包含的内容
- 任何客户的姓名、身份证号
- 客户的手机号、联系方式
- 客户的投保金额
- 具体的投保日期
- 订单信息
- 会员信息

---

## 五、数据查询示例

```sql
SELECT 
    product_name,
    product_code,
    product_type,
    company_name,
    min_amount,
    max_amount,
    min_premium,
    max_premium,
    coverage,
    description,
    details,
    image_url,
    tags,
    age_range,
    insurance_period,
    payment_period,
    waiting_period,
    exclusions,
    cooling_off_period,
    surrender_terms,
    extend_info
FROM insurance_product
WHERE id = ? AND status = 1;
```

## 六、使用场景

1. **产品展示页面**：官网、小程序产品列表
2. **产品对比**：多个产品的功能对比
3. **营销宣传**：产品手册、宣传单页
4. **产品咨询**：客服介绍产品时的参考资料
5. **AI智能体**：作为知识库供AI推荐产品
