# 医疗险核心保障权益细化

本文档旨在细化医疗险（如百万医疗险、中端医疗险等）的核心保障权益，提取通用的对比指标，以便在产品详情页及对比页中进行标准化展示。

## 一、医疗险核心指标分类表

以下表格梳理了医疗险产品中必须展示的关键权益指标：

### 1. 基础投保规则
| 指标名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| **投保年龄** | 接受投保的年龄范围 | 出生满30天-60周岁 |
| **保障期限** | 保险合同的有效时间 | 1年 |
| **最高续保年龄** | 产品可续保到的最高年龄 | 100周岁 |
| **职业限制** | 允许投保的职业类别 | 1-4类 |
| **等待期** | 首次投保后不赔付的时间段 | 30天 / 90天 |
| **犹豫期** | 投保后可全额退保的时间段 | 15天 |
| **智能核保** | 是否支持在线智能核保 | 支持 / 不支持 |

### 2. 住院医疗保障（核心权益）
| 指标名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| **一般医疗保额** | 因普通疾病或意外住院的最高报销额度 | 300万 |
| **重疾医疗保额** | 因重大疾病住院的最高报销额度 | 600万 |
| **年度免赔额** | 每年需自费的起付线（一般与重疾共享或独立） | 一般医疗1万，重疾0免赔 |
| **赔付比例** | 扣除免赔额后的报销比例（有/无社保） | 经社保100%，未经社保60% |
| **住院前后门急诊** | 住院前和出院后一定天数内的门急诊费用报销 | 前7天后30天 |
| **重疾住院津贴** | 确诊重疾住院期间的按日定额给付 | 100元/天（限180天） |
| **特殊门诊** | 肾透析、放化疗、器官移植抗排异等费用 | 含（与住院保额共享） |
| **门诊手术** | 门诊进行的手术费用 | 含 |

### 3. 特药与高端医疗保障
| 指标名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| **院外特药保障** | 医院外购买的特定药品（如抗癌药）报销 | 400万（0免赔，100%赔付） |
| **特药种类数量** | 覆盖的特药种类数量 | 120种（含CAR-T疗法） |
| **质子重离子** | 质子重离子治疗费用报销 | 600万（100%赔付，床位费限额） |
| **指定医疗器械** | 特定医疗器械费用报销 | 含（如乳房假体等） |

### 4. 增值服务
| 指标名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| **医疗费用垫付** | 住院期间保险公司垫付医药费 | 支持（不限次数） |
| **就医绿通** | 专家门诊预约、住院/手术安排 | 重疾绿通 |
| **术后护理** | 出院后的上门护理服务 | 不含 / 含1次 |
| **家庭共享免赔额** | 家庭单是否共享年度免赔额 | 支持（需2人以上投保） |

### 5. 续保与免责
| 指标名称 | 说明 | 示例值 |
| :--- | :--- | :--- |
| **续保条件** | 是否保证续保，续保是否需审核 | **保证续保20年** / 不保证续保（审核） |
| **既往症约定** | 对投保前已患疾病的赔付规定 | 既往症不赔 / 特定既往症可赔 |
| **主要免责条款** | 常见不赔付的情况 | 整容、牙科、高风险运动、遗传性疾病 |

## 二、数据结构建议 (JSON)

建议在 `insurance_product` 表的 `extend_info` (JSON字段) 中存储上述结构化数据，以便前端灵活渲染。

```json
{
  "medical_features": {
    "basic_rules": {
      "age_limit": "0-60周岁",
      "occupation_limit": "1-4类",
      "waiting_period": "30天",
      "guaranteed_renewal": "20年"
    },
    "hospitalization": {
      "general_quota": "300万",
      "critical_illness_quota": "600万",
      "deductible": "1万 (重疾0免赔)",
      "reimbursement_rate": "100%",
      "pre_post_hospitalization": "前7后30"
    },
    "special_coverage": {
      "proton_heavy_ion": true,
      "external_drugs": true,
      "special_outpatient": true
    },
    "value_added_services": [
      "医疗垫付",
      "重疾绿通",
      "家庭共享免赔额"
    ]
  }
}
```

## 三、前端展示建议

1.  **关键权益卡片**：在产品列表页，仅展示“保额”、“免赔额”、“保证续保”三个核心标签。
2.  **详细权益表格**：在产品详情页的“保障权益”部分，默认折叠展示，点击展开上述详细表格。
3.  **对比视图**：在产品对比页面，横向罗列上述指标，标红差异项。
